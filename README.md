## Проект Продуктовый помощник

### Сведения о проекте:

Проект **Foodgram** призван облегчить ежедневное приготовление пищи благодаря 
огромной базе рецептов. Также проект позволяет публиковать рецепты, 
подписываться на публикации других пользователей, добавлять понравившиеся 
рецепты в список «Избранное», а перед походом в магазин скачивать сводный 
список продуктов, необходимых для приготовления одного или нескольких 
выбранных блюд.

### Состав проекта:

Проект включает в себя:

1. Систему визуального отображения сервиса пользователю.
2. Систему хранения и обработки информации по рецептам и пользователям.
3. Систему удаленного доступа к информации, реализованную с помощью
REST архитектуры.

### Возможности проекта:

1. Проект позволяет хранить информацию:
    * о пользователях
    * о рецептах
    * о дополнительных сущностях - теги, списки покупок и т.п.
2. Проект позволяет работать с пользователями:
    * регистрировать пользователя
    * получать информацию о своем пользователе
    * получать параметры аутентификации для работы по API протоколу
3. Проект позволяет указать роль доступа пользователя:
    * пользователь
    * администратор
4. Проект позволяет добавлять рецепты:
    * в избранное
    * в список покупок
5. Проект позволяет сформировать и скачать список покупок в удобном для похода 
магазин формате

### Возможности API проекта:

1. Пользователи
    * добавление пользователя
    * получение параметров аутентификации
    * управление пользователями (просмотр)
2. Теги
    * просмотр тегов
3. Рецепты
    * добавление рецептов
    * управление рецептами (удаление, просмотр, изменение)
4. Список покупок
    * добавление рецептов в список покупок
    * управление списком покупок (удаление рецепта)
    * скачивание списка покупок в формате .txt
5. Избранное
    * добавление рецептов
    * управление избранным (удаление рецепта)
6. Подписки на авторов
    * добавление подписки
    * управление подписками (удаление, просмотр)
7. Ингредиенты
    * просмотр ингредиентов
